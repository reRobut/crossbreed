盲盒定价与机制设计方案

70% 概率：携带3个隐性基因
25% 概率：携带4个隐性基因
5% 概率：携带5个隐性基因

盲盒定价：1200 金币

60% 概率：携带5个隐性基因

5% 概率：携带8个隐性基因
