向日葵（Helianthus annuus）是一种大型菊科向日葵属植物。葵花籽常被用于食用和榨油。幼年向日葵的方向与太阳运行轨迹一致——黎明时，花的头部朝向东方，全天向西移动。然后幼年花朵会一夜之间重新调整方向面向东方，期待着早晨到来。当向日葵达到完全成熟时，它们不再跟随太阳移动，而是持续面向东方。向日葵被称为超富集植物（Hyperaccumulator），它们有着庞大的根系、生长迅速、生物量大，且具有极强的抗毒性，所以能吸收放射性物质。在核事故之后被广泛为净化目的而栽种。 |

铁线莲（Clematis florida）是毛茛科铁线莲属草质藤本植物。                                                                                                                                                                         蔷薇（Rosa）对蔷薇科蔷薇属多年生开花植物的统称，包含一百多个种以及上千个栽培品种，茎上有刺。长久以来，蔷薇就象征着美丽和爱情。西方把蔷薇花当作严守秘密的象征，做客时看到主人家桌子上方画有蔷薇，就明白在这桌上所谈的一切均不可外传，于是有了Sub   rosa，“在蔷薇花底下”这个拉丁成语。 大马士革蔷薇是伊朗、伊拉克、叙利亚、保加利亚、摩洛哥等国的国花。杂交月季是英格兰、卢森堡、美国等国的国花。                                                               |

虞美人（Papaver rhoeas）是罂粟科罂粟属草本植物。是比利时的国花。在英国与加拿大的国殇纪念日中，有佩带虞美人来纪念第一次世界大战阵亡士兵的传统，因此又称国殇虞美人花。这种花在中文里被称为“虞美人”，得名于项羽的宠妃虞姬。她的故事家喻户晓，更在京剧及同名电影《霸王别姬》中被奉为艺术经典。相传垓下之战四面楚歌，虞姬自尽殉情，其鲜血染红的土地上随后绽放出这种血红色的花朵。后人以此花命名，以此铭记她对爱情的忠贞不渝与对家国的炽热情感。                  |
